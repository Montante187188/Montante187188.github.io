Unidad 1 | actividad 03 :: tema(Parrafo)

La etiqueta align=center sirve para centrar el texto dentro de un parrafo.

La etiqueta p align="right" sirve para alinear el texto a la derecha dentro de un parrafo.

La etiqueta p align="Justify" se utiliza para justificar el texto dentro de un parrafo, lo que significa que el texto se distribuye uniformemente entre los marguenes izquierda y derecha.
