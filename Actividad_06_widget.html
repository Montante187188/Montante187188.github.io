Siglas

OSI

Modelo de interconexion de sistemas abiertos(Open Sistem Interconnection), es un marco de referencia especializado para estandarizar las funciones de un sistema de comunicacion

IP

Protocolo de Internet(Internet Protocol), se utiliza para enviar datos de un dispositivo a otro a traves de una red

TCP

Protocolo de Control de Transmision(Transmission Control Protocol), se encarga de garantizar que los datos enviados desde un dispositivo a otro lleguen de manera correcta y en orden

UDP

Protocolo de Datagrama de Usuario (User Datagram Protocol) protocolo de comunicacion mas simple que no proporciona los mismos niveles de fiabilidad y ordenacion de datos

SSH

Secure Shell es un protocolo de red criptografico utilizado para acceder de manera segura a dispositivos remotos y realizar operaciones administrativas o de transferencia de archivos

FTP

Protocolo de transferencia de archivos (File Transfer Protocol). Protocolo de red utilizado para transferir archivos entre un cliente y servidor en una red basada en TCP/IP, como internet

SMTP

Procolo Simple de Transferencia de Correo(Simple Mail Transfer Protocol) Es un protocolo utilizado para enviar y recibir correos electronicos de internet

POP

Protocolo de Oficina de Correo (Post Office Protocol). Protocolo utilizado por los clientes de correo electronico para recuperar mensajes de correo electronico desde un servidor de correo

DHCP

Protocolo de configuracion dinamica de host(Dynamic Host Configuration Protocol). Protocolo de red que permite a los dispositivos obtener automaticamente una direccion IP y otros parametros de configuracion de red sin la intervension manual de un administrador de red

DNS

Sistema de Nombres de Dominio(Domain Name System). Sistema escencial para el funcionamiento de internet, ya que traduce los nombres de dominio legibles por humanos.

HTTPS

Protocolo de Transferencia de Hipertexto(Hypertext Transfer Procol). Procolo Utilizado para la transimision de informacion en la web

ARP

Protocolo de Resolucion de Direcciones(Address Resolution Protocol). Protocolo utilizado para mapear direcciones de red(Direcciones IP) a direcciones fisicas(direcciones MAC) en una red de area local (LAN)

MAC

Control de Acceso a Medios (Media Access Control). Es la Subcapa de la capa de enlace de datos en el modelo OSI. La direccion MAC es una direccion fisica unica asignada a cada interfaz de red en un dipositivo

LAN

Red de Area Local(Local Area Network). Se refiere a una red que conectra dispositivos en un area geografica relativamente peque√±a, como un a oficina, un edificio, o un campus
